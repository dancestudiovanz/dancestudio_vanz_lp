================================================================================
VANZ 会員掲示板 — スプレッドシート コピペ用セットアップ
================================================================================


--------------------------------------------------------------------------------
【1】スプレッドシートのファイル名
--------------------------------------------------------------------------------

VANZ会員掲示板


--------------------------------------------------------------------------------
【2】シート（タブ）の名前
--------------------------------------------------------------------------------

掲示板


--------------------------------------------------------------------------------
【3】1行目ヘッダー — A1 セルを選択して貼り付け（8列）
--------------------------------------------------------------------------------

投稿日	カテゴリー	タイトル	曜日	クラス	本文	URL	画像1	画像2	画像3	画像4


--------------------------------------------------------------------------------
【4】列の説明
--------------------------------------------------------------------------------

A列 投稿日    … 日付（カレンダーから選択）
B列 カテゴリー … プルダウン: 曲 / 衣装 / おしらせ
C列 タイトル  … 一覧に表示（必須）
D列 曜日      … プルダウン: 月 / 水 / 金（空欄可）
E列 クラス    … プルダウン: 幼児 / 低学年 / can☆bang / vanZ / HIPHOP（空欄可）
F列 本文      … 展開時に表示
G列 URL       … 展開時にリンク表示
H〜K列 画像1〜4 … 画像URL


--------------------------------------------------------------------------------
【5】入力規則の設定
--------------------------------------------------------------------------------

■ A列（投稿日）A2:A400
  - 表示形式: 日付（yyyy-mm-dd）
  - データの入力規則: 日付 → 有効な日付

■ B列（カテゴリー）B2:B400
  - プルダウン: 曲 / 衣装 / おしらせ
  - 空白を許可

■ D列（曜日）D2:D400
  - プルダウン: 月 / 水 / 金
  - 空白を許可

■ E列（クラス）E2:E400
  - プルダウン: 幼児 / 低学年 / can☆bang / vanZ / HIPHOP
  - 空白を許可


--------------------------------------------------------------------------------
【6】サンプル行 — A2 セルに貼り付け（任意）
--------------------------------------------------------------------------------

2026-06-14	曲	【テスト】お知らせサンプル	月	幼児	ここに本文が入ります。	https://example.com	


--------------------------------------------------------------------------------
【7】投稿の追加・編集・削除
--------------------------------------------------------------------------------

追加 … 最下行に新しい行を入力
編集 … 該当セルを直接修正（掲示板を再読み込みで反映）
削除 … 該当行を削除
下書き … 別シート（例: 下書き）へ行を移動


--------------------------------------------------------------------------------
【8】添付画像の書き方（H列）
--------------------------------------------------------------------------------

画像なし:
（空欄）

画像1枚:
https://drive.google.com/uc?export=view&id=ファイルID

画像2枚（カンマ区切り）:
https://example.com/a.jpg,https://example.com/b.jpg


--------------------------------------------------------------------------------
【9】LP公開用「おしらせ」シート
--------------------------------------------------------------------------------

同じスプレッドシートに、シート名「おしらせ」を作成します。

1行目ヘッダー:

公開日	タイトル	本文

A列 公開日 … 日付
B列 タイトル … LPのおしらせ一覧に表示
C列 本文 … 詳細本文（改行可）


--------------------------------------------------------------------------------
【9】Script Properties（GAS の setupSecrets 参照）
--------------------------------------------------------------------------------

SPREADSHEET_ID=スプレッドシートURLの /d/ と /edit の間
MEMBER_VIEW_PASSWORD=会員用パスワード
DRIVE_FOLDER_ID=画像保存用フォルダID（任意）
