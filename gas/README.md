# VANZ 会員掲示板（GAS + スプレッドシート）

管理者が **スプレッドシート** に直接投稿し、会員が **公式サイト** の掲示板ページ（パスワード認証）で閲覧するシステムです。

## 事前準備

### スプレッドシート作成

[`spreadsheet-setup.txt`](spreadsheet-setup.txt) の【1】〜【5】を順に設定してください。

または [`Posts-template.csv`](Posts-template.csv) をインポート後、シート名を `お知らせ` に変更。

**列構成（8列）:**

```
投稿日 | カテゴリー | タイトル | 曜日 | クラス | 本文 | URL | 添付画像
```

- **カテゴリー**: 曲 / 衣装 / おしらせ
- **曜日**: 月 / 水 / 金（空欄可）
- **クラス**: 幼児 / 低学年 / can☆bang / vanZ / HIPHOP（空欄可）

## GAS へのコード配置

1. スプレッドシート → **拡張機能** → **Apps Script**
2. `gas/` フォルダ内のファイルを GAS エディタにコピー
3. `Config.gs` の `setupSecrets()` 内の値を書き換え、**1回だけ実行**
4. **デプロイ** → **新しいデプロイ** → 種類: **ウェブアプリ**
   - 実行ユーザー: **自分**
   - アクセス: **全員**
5. 発行された URL をコピー（サイトの `VITE_MEMBER_BOARD_URL` に設定）

## 投稿方法（管理者）

1. スプレッドシート `VANZ会員掲示板` → シート `お知らせ` を開く
2. 最下行に新しい行を追加
3. 投稿日・カテゴリー・タイトルなどを入力（プルダウンから選択）
4. 掲示板を再読み込みすると反映される

## 投稿の編集・削除

| 操作 | 方法 |
|------|------|
| 内容を直す | 該当セルを直接編集 |
| 掲示板から消す | 行を削除 |
| 下書きとして残す | 別シートへ行を移動 |

## 会員の閲覧

1. 公式サイト → **会員専用掲示板**
2. パスワード入力
3. お知らせ一覧を閲覧（閲覧のみ）

**一覧の見え方:**

- 折りたたみ: 投稿日 / カテゴリー / 曜日 / クラス / タイトル
- 展開（+）: 本文 / URL / 添付画像

## 公式サイト側の設定

```
VITE_MEMBER_BOARD_URL=https://script.google.com/macros/s/xxxxxxxx/exec
```

## 診断

GAS エディタから実行:

- `diagnoseBoardSetup` … スプレッドシート接続・投稿件数
- `diagnoseViewHtml` … データ読み取り確認

## 注意

- GAS のコードを更新したら **新しいデプロイ** が必要
- LINE からの投稿は **無効化** 済み（スプシのみ）
