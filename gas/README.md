# VANZ 会員掲示板（GAS + スプレッドシート）

管理者が **公式LINE** から投稿し、会員が **公式サイト** の掲示板ページ（パスワード認証）で閲覧するシステムです。

## 事前準備

### スプレッドシート作成

[`spreadsheet-setup.txt`](spreadsheet-setup.txt) の【1】〜【5】を順に設定してください。

または [`Posts-template.csv`](Posts-template.csv) をインポート後、シート名を `掲示板` に変更。

同じスプレッドシートに、LP公開用のシート `おしらせ` も作成します。

**列構成（11列）:**

```
投稿日 | カテゴリー | タイトル | 曜日 | クラス | 本文 | URL | 画像1 | 画像2 | 画像3 | 画像4
```

- **カテゴリー**: 曲 / 衣装 / おしらせ
- **曜日**: 月 / 水 / 金（空欄可）
- **クラス**: 幼児 / 低学年 / can☆bang / vanZ / HIPHOP（空欄可）

### LP公開用おしらせシート

シート名: `おしらせ`

```
公開日 | タイトル | 本文
```

## GAS へのコード配置

1. スプレッドシート → **拡張機能** → **Apps Script**
2. `gas/` フォルダ内のファイルを GAS エディタにコピー
3. `Config.gs` の `setupSecrets()` 内の値を書き換え、**1回だけ実行**
4. **デプロイ** → **新しいデプロイ** → 種類: **ウェブアプリ**
   - 実行ユーザー: **自分**
   - アクセス: **全員**
5. 発行された URL をコピー（サイトの `VITE_MEMBER_BOARD_URL` に設定）

## 投稿方法（管理者 / 公式LINE）

管理者はトークで `投稿` と送信し、案内に沿って入力します。

1. カテゴリー選択（曲 / 衣装 / おしらせ）
2. タイトル入力（例: 4月分衣装）
3. 曜日選択（月 / 水 / 金 / 空欄）
4. クラス選択（幼児 / 低学年 / can☆bang / vanZ / HIPHOP / 空欄）
5. 本文入力（URLは本文内にそのまま貼り付け）
6. 画像送信（任意、最大4枚）
7. `確認` → `送信` で投稿確定（`キャンセル` で破棄）

投稿確定後にスプレッドシートへ自動出力され、掲示板に反映されます。

## 投稿の編集・削除

| 操作 | 方法 |
|------|------|
| 内容を直す | シート `掲示板` の該当セルを直接編集 |
| 掲示板から消す | 該当行を削除 |
| 下書きとして残す | 別シートへ行を移動 |

## 会員の閲覧

1. 公式サイト → **会員専用掲示板**
2. パスワード入力
3. お知らせ一覧を閲覧（閲覧のみ）

**一覧の見え方:**

- 折りたたみ: 投稿日 / カテゴリー / 曜日 / クラス / タイトル
- 展開（+）: 本文（URLは本文内リンク）/ 添付画像

## 公式サイト側の設定

```
VITE_MEMBER_BOARD_URL=https://script.google.com/macros/s/xxxxxxxx/exec
VITE_ANNOUNCEMENTS_URL=https://script.google.com/macros/s/xxxxxxxx/exec?page=announcements
```

## 診断

GAS エディタから実行:

- `diagnoseBoardSetup` … スプレッドシート接続・投稿件数
- `diagnoseViewHtml` … データ読み取り確認

## 注意

- GAS のコードを更新したら **新しいデプロイ** が必要
- LINE投稿を使うため、`setupSecrets()` 後に `LINE_CHANNEL_SECRET` / `LINE_CHANNEL_ACCESS_TOKEN` / `ADMIN_LINE_USER_ID` を Script Properties に設定してください
- 画像保存には `DRIVE_FOLDER_ID` が必要です（未設定時は画像アップロード不可）
